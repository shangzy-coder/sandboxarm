# PAL-Math

This dataset refers to the Deepseek Coder's [PAL-Math](https://github.com/deepseek-ai/DeepSeek-Coder/tree/main/Evaluation/PAL-Math) implementation, which enables LLMs to solve math problems with python code. Seven datasets are included: GSM8k, MATH, GSM-Hard, SVAMP, TabMWP, ASDiv, and MAWPS.
